Spend wisely

Scale Insights subscription plan charges based on the number of automated ASINs you need.
1
Automated ASINs are based on what you automate, not the number of ASINs you have

If you have 1000 ASINs but only want automation for 3 ASINs, then you only need 3 automated ASINs.

2
Automated ASINs are based on the ad group's associated ASIN

If you only have the ASIN B000000001 in an ad group, Scale Insights will set that as the associated ASIN.
If you enable automation for the B000000001, you can automate this ad group and any other ad groups associated with this ASIN.

To automate an ad group with multiple variations, you must manually set an associated ASIN for the ad group, then enable automation for the associated ASIN.

To automate multiple ad groups associated with the same ASIN, you only need 1 automated ASIN for that ASIN.

3
One automated ASIN allows you to automate one ASIN in one marketplace

To automate 1 ASIN in the US and the same ASIN in CA, you need 2 automated ASINs.
